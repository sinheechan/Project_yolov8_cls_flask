# Project_yolov8_cls_flask


## Object

Flask는 웹 애플리케이션을 구축하는 데 필요한 핵심 기능을 제공합니다.

Ultralytics의 YOLOv8 모델은 객체 감지를 위한 딥러닝 알고리즘으로 이미지나 비디오에서 객체를 감지하고 분류하는 데 사용됩니다.

본 모델은 Flask를 사용하여 YOLO 모델을 웹 애플리케이션으로 배포하는 과정을 테스트합니다.
